# Design Thinking

- Enfoca la experiencia en descubrir productos, comparar opciones y comprar sin fricción.
- Prioriza claridad, confianza y rapidez en móvil.
- Diseña para clientes que buscan categorías, stock, precio final y checkout simple.

# UI/UX Principles

- Una acción primaria por pantalla.
- Información esencial primero, detalles después.
- Componentes consistentes para cards, botones, filtros y modales.
- Precio y stock siempre visibles o fáciles de consultar.

# Visual Guidelines

- Usa una paleta deportiva y limpia con un color acento fuerte para CTAs.
- Mantén jerarquía clara con tipografía legible y buen contraste.
- Aplica una grilla simple, cards ordenadas y espacios amplios.
- Usa imágenes grandes y recortes consistentes para productos.

# Interaction Design

- Filtros accesibles, búsqueda clara y navegación rápida entre categorías.
- Estados visibles de carga, vacío y error.
- Checkout corto, con resumen de compra antes de confirmar.
- Microinteracciones suaves para hover, selección y confirmación.

# Best Practices

- Diseña accesible: foco visible, labels claros y botones con tamaño táctil correcto.
- Piensa en responsive desde el inicio.
- Mantén el copy breve y directo.
- Evita adornos que resten lectura o distraigan del producto.
